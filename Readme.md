### Iniciando projeto 
- yarn init -y

### Instalando o Express
- yarn add express

### Instalando o Nodemon, como dependência
- yarn add nodemon -D

### Intalando o Sucrase
- yarn add sucrase -D

### Validar campo
- yarn add -S yup

### Criptogrfia

- yarn add bcryptjs

### Json WebToken

- yarn add install jsonwebtoken

### Paginação com mongoose Paginate-v2

- yarn add mongoose-paginate-v2
